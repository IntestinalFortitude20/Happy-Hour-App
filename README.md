# The Happy Hour App

Chicago Happy Hour & Event Tracker / Finder
A robust community platform for discovering local hospitality events and live music.

## Description

This application is a comprehensive directory for the Chicago "after-hours" scene. Moving beyond a simple database, it tracks a variety of recurring community events including Jazz Jams, Trivia nights, Bingo, and Live Music performances. It serves as a bridge between local venues looking to fill seats and musicians or patrons looking for engagement.

## Why I Built It

Having worked in the restaurant industry, I understand the challenge of driving foot traffic on "off-peak" nights. I also know how difficult it can be for local musicians to find consistent jam sessions or performance opportunities. I though to address both of these issues with a single solution.

Drawing on Behavioral Psychology and my background as a Program Director, I built this app to reduce the "decision fatigue" associated with finding local entertainment. By categorizing events by type and location, the app creates a low-friction pathway for users to support local businesses and the arts simultaneously.

## Tech Stack

**Frontend:** React & Vite
**State Management:** React Hooks for dynamic event filtering
**Database/API:** Custom JSON structure (transitioning to a full Backend/Database)
**Design:** Mobile-First CSS with a focus on high-contrast, "night-mode" readability
